# ENG-004：发布前 checklist（复制到 Release PR / 打 tag 前自检）
- [ ] `dev` → `main` PR 已合并且 CI 全绿（`run-ci-full-suite.sh`）
- [ ] 已运行 `./tests/run-eng-release-precheck.sh`（或 `SHU_RELEASE_TAG=vX.Y.Z` 带 tag 校验）
