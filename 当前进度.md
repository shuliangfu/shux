| 波 | 结果 |

| **codegen M2 dep 共 emit 根因下钻** | 复现 force-regen：option C 仅 `core_option_none_i32` 体 + 其余 extern，或更差时无任何函数体/`main`。对照 force C 中 `codegen_x_ast`：**`while (i < num_funcs)` 的 per-func emit 被 pin 嵌进 `if (i==0)`（或 `if (num_top_level_lets>0)`）**，随后 `return 0` 仍在 while 内 → 只 emit 第一函数或 0 函数。**权威产生点 = pin seed 编译 Cap `codegen.x` 时的大块 if 后继语句嵌套错误**（非 option/fmt 链接门控）。 |

| **结论** | M2 链上矩阵收口须修 **pin 对大 if 后继语句的 brace/块嵌套**（seed + `.x` 同 commit），或找到 pin 能完整 emit 且 while 不被吞的 Cap 结构；**禁**在 Cap 上硬拆导致 53KiB 假 Cap。**seed pin / tip L4 未动**。 |
