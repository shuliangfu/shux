## 0. 读表前：三层目标别混

| 层 | 名称 | 当前 | 终局判据（建议） |
|----|------|------|------------------|
| **L0 架构** | 产品不编手写 `src/**/*.c`；seed 化；F-09 src=0 | ✅ 基本完成 | 保持 G05 51 + STRICT + noinc + rv42 + lang-unsafe 双绿 |
| **L1 真迁** | 逻辑在 `.x`；seed 折叠 thin `_impl`；可审计 | 🟡 ~40–55%（折叠大进；真 .x AST 仍欠） | 见 §1 闭合判据 |
| **L2 执行** | 产品 **链 `.x` 编出的 `.o`**，而非手写 seed C | ⬜ 未开始 | G05 热路径 objs 由 `.x` 工具链产出 |

**关键现实**：今天即使 `.x` 写了真逻辑，**产品仍 `cc seeds/*.from_x.c`**。L1 是「逻辑主权」；L2 才是「运行时零手写 C」。

## 5. 每日「继续下一步」操作卡

1. 打开本表 **Wave 当前未勾第一行**。  
2. 实现：`.x` 真逻辑 + seed 同语义/折叠 `_impl`（不扩 scope）。  
3. `自举进度.md`：只改 **更新时间头** + **§7 row 11** → `f-N`。  
4. macOS：`g05_prepare_and_relink` → STRICT → noinc → rv42 → lang-unsafe。  
5. `git commit` + `push`。  
6. Ubuntu：`fetch/reset` → 清 `std/sort/sort.o` / crypto glue 如需 → 同门禁。  
7. 本表勾选该步；若触达里程碑，刷新 §2 数字（wrap 计数脚本）。

### 5.1 wrap 计数（刷新 §2 用）

```bash
# 在仓库根；统计 thin pub→_impl
python3 - <<'PY'
import re, pathlib
pat=re.compile(r'\n[\w\s\*]+?\s+(\w+)\s*\([^)]*\)\s*\{\s*\n\s*\{\s*\n\s*(?:return\s+)?(\w+_impl)\s*\(', re.M)
n=0
for p in pathlib.Path('compiler/seeds').glob('*.from_x.c'):
    n += len(pat.findall(p.read_text(errors='ignore')))
print('thin_wrappers', n)
PY
```
