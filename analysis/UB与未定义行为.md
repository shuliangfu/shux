# 自举子集内未定义行为（UB）清单与约定

> 「超越 C」的前提之一是语言层面减少 UB，使优化与推理可依赖。本文档列出**自举所需语言子集内**的未定义行为清单，并约定「默认行为」或「明确为未定义」；便于自举后编译器与标准库不依赖未文档化的 UB。对应《自举前-目标与缺口分析》目标一、UB 项。

---

## 一、目标与范围

- **范围**：自举前以**文档化**为主，列出自举子集中可能涉及 UB 的构造及约定。
- **目的**：为优化与安全推理打基础；自举后编译器与标准库可据此避免依赖未文档化行为。

---

## 二、自举子集内 UB 清单与约定

以下为自举所需语言子集内可能涉及未定义行为的类别；每项给出**约定**（默认行为或明确为 UB）。

| 类别 | 说明 | 约定 |
|------|------|------|
| **整数溢出** | 有符号整数运算（+、-、*等）结果超出类型范围 | **明确为未定义**。自举后若需可预测包装语义，可使用 wrapping 或显式检查；当前生成 C 时依赖 C 的溢出为 UB，不做额外保证。 |
| **除零 / 取模零** | `/` 或 `%` 的除数为 0 | **明确为未定义**。实现可在运行时插入检查（panic）以缩小 UB 范围；自举前可约定「除零为未定义，实现可选择 panic」。 |
| **空指针解引用** | 对空指针（*T 为 0 或 NULL）做 `*p` 或 `p->field` | **明确为未定义**。与 C 一致；不承诺检测。 |
| **越界访问** | 数组下标或切片索引超出 [0, len) | **明确为未定义**。实现可插入边界检查（panic）以缩小 UB；自举前约定「越界为未定义，实现可选择检查」。 |
| **悬垂指针 / use-after-free** | 使用已释放或失效的指针或切片 | **明确为未定义**。语言不提供生命周期静态检查时，由程序员保证。 |
| **数据竞争** | 多线程下对同一内存的无同步并发写或读写 | **明确为未定义**。自举子集若尚未引入线程模型，可预留；若与 C 互操作多线程，约定与 C11 内存模型一致。 |
| **未初始化读取** | 读取未赋值的变量或结构体字段 | **明确为未定义**。类型检查可要求显式初始化以减少此类情况。 |
| **非法类型转换** | 未约定的裸指针与整数、或不同类型指针间的转换 | **明确为未定义或实现定义**。FFI 中与 C 的转换以《与 C 互操作》与 ABI 文档为准。 |

---

## 三、默认行为（非 UB）的约定

以下行为在自举子集内**不作为未定义**，便于优化与推理：

- **无符号整数运算**：按 2^N 取模，与 C 一致。
- **布尔与逻辑短路**：`&&` / `||` 短路求值；逻辑运算结果 0/1。
- **结构体与切片布局**：按《内存契约》与《ABI 与布局》约定；符合约定则无布局 UB。

---

## 四、与其它文档的对应

- **《自举前-目标与缺口分析》**：UB 项为自举前建议；本文档满足「自举子集内 UB 清单与约定」。
- **《与 C 互操作》**：FFI 边界上的行为与 C 一致；C 侧 UB 在跨边界时仍为 UB。
- **《内存契约》**：布局约定减少「因布局歧义」导致的潜在 UB。

## 六、有符号溢出策略（P1-2，typeck / codegen 对齐）

| 场景 | 策略 | 实现现状 |
|------|------|----------|
| **语言语义** | 有符号 `+` `-` `*` 结果超出类型范围 → **未定义行为（UB）** | 与 §二 一致；不承诺 wrapping |
| **无符号运算** | 按 \(2^N\) 取模（**非 UB**） | 与 C 一致；见 `tests/ub/unsigned_wrap_ok.sx` |
| **编译期常量（CTFE）** | 字面量与常量折叠在 **i64** 宽域计算；赋值到较窄类型时须可表示 | lexer/i64 CTFE 回归见 `run-i64-ctfe-gate.sh` |
| **Release 运行时** | **不**插入有符号溢出检查；生成 C 时依赖 C 语义（仍为 UB） | 零成本；与「性能比肩 C」一致 |
| **Debug / 未来** | 可选 `-fshux-check-overflow` 或显式 `wrapping_add` 等 API → panic / 可预测 wrapping | **未实现**；新增须改 typeck + codegen 并扩 gate |

**验收**：`tests/run-signed-overflow-gate.sh`（manifest + 无符号 wrap 烟测）；除零/越界见 `tests/run-ub.sh`。
