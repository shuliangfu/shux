# 类型定义

## 内建类型

| 类型 | 说明 |
|------|------|
| `i32` | 32 位有符号整数 |
| `i64` | 64 位有符号整数 |
| `u8` | 8 位无符号整数（字节） |
| `u32` | 32 位无符号整数 |
| `u64` | 64 位无符号整数 |
| `usize` | 与指针同宽的无符号整数（长度、下标等） |
| `isize` | 与指针同宽的有符号整数 |
| `bool` | 布尔 |
| `f32` | 32 位浮点 |
| `f64` | 64 位浮点 |
| `i32x4` / `i32x8` / `i32x16` | 4/8/16 车道 i32 向量（SIMD；16 对应 512-bit/AVX-512） |
| `u32x4` / `u32x8` / `u32x16` | 4/8/16 车道 u32 向量（SIMD） |

## 复合类型

### 指针

- 裸指针：`*T`，如 `*i32`。

### 数组

- 固定长数组：`[N]T`，如 `[10]i32`。`N` 为编译期常量。

### 切片

- 切片：`[]T`，如 `[]i32`。语义为「指针 + 长度」；有 `.len` 等成员。

### 结构体

- 定义：`struct Name { field: Type; ... }`
  - **字段之间用分号 `;` 分隔**（不是逗号）。例如：`struct Point { x: i32; y: i32 }`。若误写为逗号会导致解析错误。
- **无填充布局**：`struct packed Name { field: Type; ... }`，与 C `__attribute__((packed))` 一致，无隐式 padding、对齐为 1。
- 支持泛型：`struct Name<T> { ... }`
- 字面量：`Name { field: expr, ... }`（字面量里字段之间用**逗号**，与定义不同）
- 字段访问：`value.field`

### 枚举

- 定义：`enum Name { VariantA, VariantB, ... }`（无负载枚举）；**变体之间用逗号 `,` 分隔**（与 struct 字段用分号不同）。
- 变体引用：`Name.VariantA`（类型名 + 变体名）
- 在 `match` 中可对整型或枚举值按变体分支。

### Trait 与 impl

- Trait 定义：`trait TraitName { function method(...): RetType; ... }`；**方法声明之间用分号 `;` 分隔**（与 struct 字段一致，不要用逗号）。
- 实现：`impl TraitName for TypeName { function method(self: TypeName, ...) { ... } }`
- 方法调用：`value.method(args)`，由 typeck 解析到对应 impl 中的函数。

## 类型注解

- 变量/常量：`let x: i32 = 0;`、`const N: i32 = 10;`
- 函数参数与返回值：`function f(x: i32, y: bool): i64 { ... }`
- 泛型参数：`function f<T>(x: T): T { ... }`、`struct S<T> { ... }`

## 联合类型（Union）

变量、参数、返回值可使用 **`|` 联合** 标注「允许多种类型之一」：

```su
let x: i32 | i64 = 42;        // 初值为 i32 字面量，符号表存 i32
let y: u32 | u64 = 7 as u32;  // 存具体成员类型
function take(v: i32 | i64): i32 { return v as i32; }
```

语义要点：

| 场景 | 行为 |
|------|------|
| `let` 带联合注解 | symtab 存**初值的具体类型**（非整个 union） |
| 字面量 / 隐式转换 | 须能 assign 到联合的某一成员（如非负 `i32` 字面量可赋给 `u32 \| u64`） |
| 函数形参为 union | 调用实参须 assignable 到该 union；**不做**按调用点单态化 |
| 多类型 API | 优先用**函数重载**（见下），而非 union 形参 + 别名 |

联合类型用于「同一变量在不同分支可能是几种类型之一」；需要按实参分派时，用重载更清晰。

## 函数重载（Overload）

同一模块内可定义**多个同名函数**，形参列表（类型与个数）不同：

```su
function pick(x: i32): i32 { return x + 1; }
function pick(x: i64): i32 { return x as i32 + 2; }

function main(): i32 {
  pick(10);           // → i32 重载
  pick(20 as i64);    // → i64 重载
  return 0;
}
```

调用规则：

| 规则 | 说明 |
|------|------|
| 分派依据 | 实参已 typecheck 后的类型，与候选形参做 assignability 匹配 |
| 优选 | **精确类型匹配**优先于隐式拓宽（如 `u32` 实参优先选 `u32` 形参，而非 `i32`） |
| 歧义 | 多个候选同分 → 报错 `ambiguous call to overloaded function` |
| 未匹配 | 报错 `no matching overload for 'name'` |
| `main` | 仍须唯一，不可与其他 `main` 重载 |
| 跨模块 | `import` 后 `mod.func(...)` 同样按重载解析；codegen 对多 overload 做 C 符号 mangling（如 `format_2_i32_u32`） |

标准库示例（`std.fmt`）：

```su
const fmt = import("std.fmt");
fmt.format_2(buf, cap, 10, 20);              // i32, i32
fmt.format_2(buf, cap, 1 as usize, 2 as usize); // usize, usize
```

无 `format_2_i32_i32` 等兼容别名；同名即 API。
