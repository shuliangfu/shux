# 表达式与运算符

## 字面量

| 种类 | 示例 | 说明 |
|------|------|------|
| 整数 | `0`、`42` | 默认类型可由上下文推断（如 i32） |
| 浮点 | `0.0`、`1.5`、`.5`、`1e2`、`1.5e-1` | 含小数点或指数为浮点 |
| 布尔 | `true`、`false` | |

## 运算符优先级（从高到低）

1. **一元**：`-`（取负）、`~`（按位取反）、`!`（逻辑非）
2. **乘除模**：`*`、`/`、`%`，左结合
3. **加减**：`+`、`-`，左结合
4. **移位**：`<<`、`>>`，左结合
5. **比较**：`<`、`<=`、`>`、`>=`
6. **相等**：`==`、`!=`
7. **按位与**：`&`
8. **按位异或**：`^`
9. **按位或**：`|`
10. **逻辑与**：`&&`，短路
11. **逻辑或**：`||`，短路
12. **三元**：`cond ? then_expr : else_expr`
13. **赋值**：`=`，右结合（如 `a = b = c`）

## 二元运算符一览

- 算术：`+`、`-`、`*`、`/`、`%`
- 移位：`<<`、`>>`
- 按位：`&`、`|`、`^`
- 比较：`<`、`<=`、`>`、`>=`、`==`、`!=`
- 逻辑：`&&`、`||`

条件（if/while/for 条件、三元、逻辑运算）要求 `bool`，禁止整型隐式当布尔使用。

## 其它表达式

- **括号**：`( expr )`，用于覆盖优先级。
- **赋值**：`left = right`；在 for 的 step 中常用（如 `i = i + 1`）；left 须为可赋值左值。
- **函数调用**：`name(args)` 或 `name<TypeArgs>(args)`（泛型调用）。
- **方法调用**：`receiver.method(args)`。
- **字段访问**：`value.field`；枚举变体：`TypeName.VariantName`。
- **下标**：`base[index]`，base 为数组或切片；`index` 为 `i32`/`isize`/`i64` 时可隐式用于数组/切片/指针（等价于 `usize`）。
- **结构体字面量**：`TypeName { field: expr, ... }` 或 `let T = { field: expr, ... }`；字段之间可用 `,` 或 `;`。
- **match**：`match expr { lit | lit => result; _ => default; }`；同一分支可写 `1 | 2 | 3 => expr`。
- **数组字面量**：`[ e1, e2, ... ]`。

## 上下文类型（期望类型传播）

在以下位置，编译器按**期望类型**对字面量与部分表达式做编译期 coerce，通常可省略 `as`：

| 语境 | 示例 |
|------|------|
| `let` / `const` 初值 | `let n: u32 = 0` |
| 赋值 `=` / 复合赋值 | `total = 0`（`total: isize`） |
| `return` | `return -1`（返回 `i64`）；`return 0`（返回 `*T`） |
| 函数实参 | `poll(fds, n, to)`（`n: i32` → `u64` 形参） |
| struct 字段初值 | `{ revents: 0 }`（字段 `i16`） |
| 指针比较 | `p == 0`（`p: *T`） |
| 取址实参 | `foo(&buf[0])`（形参 `*u8`，无需 `as *u8`） |

仍须显式 `as` 的场景：不同 ABI struct 名义类型、unsafe 边界、编译器无法证明安全的转换。
