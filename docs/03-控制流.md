# 控制流

## 条件：if / else

- **if 表达式 / 语句**：`if cond { then } else { else }`，有返回值时 then/else 类型一致。
- **括号可选**：`if cond { … }` 与 `if (cond) { … }` 均可（wave361）；条件后直接跟 `{`。if 后还可接其它语句（wave362 修 bare `if` 后 IDENT 语句挂死）。
- **else 可选**：若省略 else，则按「否则为 0」等规则处理（具体以 typeck 为准）。
- 条件 `cond` 须为 `bool` 类型（禁止整型隐式转 bool）。
- **IDENT + `{` 消歧**：条件里的 `b { body }` 中 `b` 为变量；`Type { field: e }` 结构体字面量在条件外仍可用。
- **FIELD_ACCESS + `{` 消歧**（wave370）：条件末位为字段/枚举变体时，`if a.x {…}` / `if 1 == b.x {…}` / `if c == Color.Red {…}` / `while b.x != 0 {…}` 合法；`{` 留给 then/while 体。限定 struct 字面量 `Module.Type { field: e }`（如 `process.SpawnIo { … }`、`heap.PageMmapHeap { … }`）在值位置仍解析为 STRUCT_LIT（与 IDENT 共用 G.7 `lbrace_looks_like_block` / empty prefer-block）。
- **空体**（wave364）：`if b {}` / `while b {}` 合法。空命名结构体字面量 `S {}` 在**值位置**（后接 `;` / `,` / `)` / `.` / `as`）仍解析为 struct lit；条件里若需空 struct 作条件，请写括号形式 `if (S {})`（少见）。
- **STRUCT_LIT 末字段空 Type {}**（wave367）：`A { x: 42, e: E {} }` 合法。field 值解析深度下，空 `Type {}` 后的外层 `}` 仍保持 struct lit（不与 bare if/while 空体 prefer-block 冲突）。

示例：

```su
let x: i32 = if a > 0 { 1 } else { 2 };
let y: i32 = if (a > 0) { 1 } else { 2 };
```

## 循环：while / loop / for

- **while**：`while cond { body }`（括号可选，同 if），条件须为 `bool`。
- **loop**：`loop { body }`，等价于 `while true { body }`。
- **for**：C 风格 `for (init; cond; step) { body }`；init/cond/step 均可省略（cond 空表示恒真）。

在循环体内可使用：

- **break**：跳出当前循环。
- **continue**：进入下一轮循环。

示例：

```su
let n: i32 = 0;
for (let i: i32 = 0; i < 10; i = i + 1) {
  n = n + i;
}
```

## 多分支：match

- 语法：`match expr { pat => result; ... }`
- 分支形式：
  - 整数字面量：`1 => 2;`
  - 枚举变体：`Name.Variant => expr;`
  - 通配：`_ => expr;`（其它情况）
- 每个分支以分号结尾；match 整体为表达式，有类型。
- **语句位置**（wave371）：函数/块中可写独立 `match expr { … }`（不必包在 `return`/`let` 里）；臂结果可为值、赋值或 `return expr`（与表达式 match 同一 AST）。
- host-C：含 `return` 臂的 match 在 final 位置展开为带比较的三元值（`return` 只保留操作数）；臂 `return` 之后仍有可达语句时 early-return 语义 soft leave-off。

示例：

```su
return match x { 1 => 2; 2 => 3; _ => 0; };

// statement form (mid-body or final)
match x {
  1 => return 2;
  _ => return 0;
}
```

## 延迟执行：defer

- **defer 块**：`defer { block }`，在**当前块退出时**逆序执行所有已注册的 defer 块。
- 用于清理资源、保证执行顺序等。

## 跳转：label 与 goto

- **标签**：`label_name: stmt`
- **goto**：`goto label_name;`，跳转到同函数内该标签处。
- 与 return 等组合可表达显式控制流。

## 返回与终止

- **return**：`return;` 或 `return expr;`。有返回类型的函数须显式 return 带值；**void** 函数用 `return;`。入口 **main** 可为 `i32`/`i64`（退出码）或 **`void`（隐含 exit 0）**。
- **panic**：`panic;` 或 `panic(expr);`，终止程序；可选打印消息（如整数）。用于不可恢复错误。
