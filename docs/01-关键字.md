# 关键字

Shu 中以下标识符为**保留关键字**，不可作为变量名、函数名或类型名使用。

## 声明与定义

| 关键字 | 用途 |
|--------|------|
| `function` | 函数定义 |
| `let` | 变量声明（块内，须带类型与初始值） |
| `const` | 常量声明（块内或顶层，须为常量表达式） |
| `struct` | 结构体定义 |
| `packed` | 无填充结构体（与 C `__attribute__((packed))` 一致，用于 `struct packed Name { ... }`） |
| `enum` | 枚举定义（无负载枚举） |
| `trait` | trait（接口）定义 |
| `impl` | 为类型实现 trait（impl Trait for Type { ... }） |
| `import` | 导入模块（如 `import("core.types");`） |
| `extern` | 声明外部 C 函数（无体，由链接器解析，如 `extern function foo(x: i32): void;`） |

## 控制流

| 关键字 | 用途 |
|--------|------|
| `if` | 条件分支（可作表达式） |
| `else` | if 的否则分支 |
| `while` | 条件循环 |
| `loop` | 无限循环（等价 `while true`） |
| `for` | C 风格 for(init; cond; step) 循环 |
| `break` | 跳出当前循环 |
| `continue` | 进入下一轮循环 |
| `return` | 从函数返回（可带返回值） |
| `panic` | 终止程序，可选带消息（如 `panic` 或 `panic(expr)`） |
| `match` | 多分支匹配（整数字面量、枚举变体、`_` 通配） |
| `defer` | 块退出时逆序执行（`defer { ... }`） |
| `goto` | 跳转到标签（须与 `label:` 配合） |
| `unsafe` | 标记不安全代码块（`asm!`、裸指针解引用等须在其中） |

## 特殊标识符

| 关键字 | 用途 |
|--------|------|
| `self` | 方法接收者（在 impl 块内的方法中） |
| `_` | match 中的通配模式，表示“其它情况” |

## 内联汇编关键字

| 关键字 | 用途 |
|--------|------|
| `asm` | 内联汇编入口（`asm!(...)` 须在 `unsafe {}` 内） |
| `asm goto` | 带标签跳转的内联汇编（`asm goto!(...)` 须在 `unsafe {}` 内） |

## 类型名（内建，亦为保留）

以下为**内建类型名**，不能作为变量/函数名使用，但可作为类型注解：

- 整数：`i32`、`i64`、`u8`、`u32`、`u64`、`usize`、`isize`
- 浮点：`f32`、`f64`
- 布尔：`bool`
- 向量（SIMD）：`i32x4`、`i32x8`、`i32x16`、`u32x4`、`u32x8`、`u32x16`

字面量：

- `true`、`false`（布尔）
- 整数字面量（如 `0`、`42`）
- 浮点字面量（如 `0.0`、`1.5`、`.5`、`1e2`）

## 注释

Shu 支持三种注释方式，词法阶段会整体跳过、不产出 token：

| 方式 | 写法 | 说明 |
|------|------|------|
| 行注释 | `// ...` | 从 `//` 到该行行尾 |
| 块注释 | `/* ... */` 或 `/** ... **/` | 从 `/*` 到最近的 `*/`，可跨行 |
| 井号行注释 | `# ...` | 从 `#` 到该行行尾（与预处理指令区分：预处理先执行，未被消费的 `#` 行视为注释） |

书写格式（缩进、分号、块与 `}` 等）见 [08-语法规范](./08-语法规范.md)。

## 预处理（条件编译）

在词法之前执行的条件编译，按行处理以 `#` 开头的指令（未被识别的 `#` 行视为注释，见上文）：

| 指令 | 说明 |
|------|------|
| `#if SYMBOL` | 若 SYMBOL 在编译器提供的定义列表中（如通过 `-D SYMBOL`），则保留下一块直到 `#else`/`#elseif`/`#endif`；否则跳过。 |
| `#elseif SYMBOL` | 前述分支被跳过时，若 SYMBOL 已定义则保留本块直到 `#else`/`#elseif`/`#endif`。 |
| `#else` | 前述分支均未保留时，保留本块直到 `#endif`。 |
| `#endif` | 结束当前 `#if` 块。 |

支持嵌套。除单标识符外，亦支持 `target_os == "linux"`、`target_os = "linux"`、`!=` 等与 `#[cfg(...)]` 对齐的表达式（见 [09-条件编译](./09-条件编译.md)）。

编译时通过 `-D SYMBOL` 传入定义，未传入的 SYMBOL 视为未定义。

**属性式条件编译** `#[cfg(...)]` 在 parse 前剪枝顶层项与函数内块/语句，与 `#if` 共用条件求值；选用原则与示例见 [09-条件编译](./09-条件编译.md)。
