# Shux VSCode 扩展 — 打包与安装说明

本文档记录扩展的**编译、打包顺序**以及**在 node 不在 PATH 时的处理方式**，便于本机或 CI 复现。

---

## 一、操作顺序（推荐）

按以下顺序执行，在扩展目录下进行：

1. **安装依赖**（首次或 package.json 变更后）
   ```bash
   cd editors/vscode
   npm install
   # 或
   yarn
   ```

2. **编译 TypeScript**
   ```bash
   npm run compile
   # 或
   yarn compile
   ```
   产出在 `out/extension.js`、`out/extension.js.map`。

3. **打包 VSIX**（不再自动执行 compile，需先完成第 2 步）
   ```bash
   npx vsce package
   ```
   生成 `vscode-shux-0.1.0.vsix`（版本号以 package.json 为准）。

4. **安装**
   - 在 VSCode 中：**扩展** → **⋯** → **从 VSIX 安装...** → 选择生成的 `.vsix` 文件。
	 - 在菜单里选 “从 VSIX 安装...”（或 Install from VSIX...）。
