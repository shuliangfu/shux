# Vim 插件功能增强路线图

> **当前状态**：基础高亮 + 缩进 + 文件类型检测（4 个运行时文件）
> **目标**：达到 Rust/Go 语言 Vim 插件的体验水平（高亮 + 补全 + 诊断 + 跳转 + 格式化 + 折叠）
> **策略**：自举完成前完善"无编译器依赖"的功能（补全/折叠/文本对象），自举后接入 `shux-c` 做诊断/跳转/格式化

## 二、功能增强清单（按优先级）

### P0：无编译器依赖，立即可做

#### 2.1 语法折叠

**现状**：无折叠支持
**目标**：按语法结构折叠 function/struct/enum/block/match

**实现**：`ftplugin/shux.vim` 中开启 `foldmethod=syntax`，`syntax/shux.vim` 中定义 fold region

```vim
" ftplugin/shux.vim
setlocal foldmethod=syntax
setlocal foldlevel=1
setlocal foldtext=ShuxFoldText()

function! ShuxFoldText() abort
  let line = getline(v:foldstart)
  let sub = substitute(line, '/\*\.\*\*/\s*', '', 'g')
  return sub . ' ... ' . (v:foldend - v:foldstart) . ' lines'
endfunction
```

```vim
" syntax/shux.vim 中添加 fold region
syn region shuxFunctionFold start="function\>" end="^\}" fold transparent keepend
syn region shuxBlockFold start="{" end="}" fold transparent
```

#### 2.2 关键字补全（omnifunc）

**现状**：无补全
**目标**：`<C-x><C-o>` 触发关键字/类型名/函数名补全

**实现**：`autoload/shux.vim` + `ftplugin/shux.vim`

```vim
" ftplugin/shux.vim
setlocal omnifunc=shux#Complete

" autoload/shux.vim
let s:keywords = [
  \ 'function','let','const','if','else','while','for','loop',
  \ 'break','continue','return','defer','match','struct','enum',
  \ 'trait','impl','type','import','extern','export','packed',
  \ 'soa','align','unsafe','region','with_arena','as','goto',
  \ 'await','self','panic','true','false'
  \ ]
let s:types = [
  \ 'i8','i16','i32','i64','isize',
  \ 'u8','u16','u32','u64','usize',
  \ 'f32','f64','bool','void'
  \ ]

function! shux#Complete(findstart, base) abort
  if a:findstart
    let line = getline('.')
    let start = col('.') - 1
    while start > 0 && line[start - 1] =~ '\w'
      let start -= 1
    endwhile
    return start
  endif
  let result = []
  for kw in s:keywords + s:types
    if kw =~ '^' . a:base
      call add(result, {'word': kw, 'kind': 'k'})
    endif
  endfor
  return result
endfunction
```

**增强**：用 `syntax` 命令扫描当前 buffer 的所有 `function name` / `struct Name` / `let name`，加入补全候选（buffer-local 符号补全）。

#### 2.3 文本对象

**现状**：无
**目标**：`dif`/`daf` 删除函数，`dib`/`dab` 删除 block

**实现**：`autoload/shux.vim`

```vim
" function 文本对象：af（含 function 行），if（仅 body）
function! shux#TextObject(type) abort
  " 向上查找 function 声明行
  let start = search('^\s*\(export\s\+\)\?function\>', 'bnW')
  if start == 0 | return | endif
  " 向下查找匹配的 }
  let end = searchpair('{', '', '}', 'nW')
  if end == 0 | return | endif
  if a:type ==# 'af'
    execute 'normal! ' . start . 'GV' . end . 'G'
  else
    execute 'normal! ' . (start + 1) . 'GV' . (end - 1) . 'G'
  endif
endfunction
```

#### 2.4 注释切换

**现状**：`//` 注释格式已设置，但无快速切换

**实现**：利用 Vim 内置 `commentstring`，vim-commentary / nerdcommenter 等插件自动识别。只需确保 `commentstring=//\ %s`（已设置）。

#### 2.5 matchit 增强

**现状**：基础括号匹配
**目标**：`%` 在 `if/else/match/for/while` 之间跳转

**实现**：`ftplugin/shux.vim`

```vim
" matchit 配置：关键字配对跳转
let b:match_words = '\<if\>:\<else\>,'
  \ . '\<for\>:\<break\>,'
  \ . '\<while\>:\<break\>,'
  \ . '\<loop\>:\<break\>,'
  \ . '\<match\>:\<=\>:,\@<=\s*,'
  \ . '{:},(:),[:],'

```

```vim
  au User lsp_setup call lsp#register_server({
    \ 'name': 'shux',
    \ 'cmd': {server_info->['shux', '--lsp']},
    \ 'whitelist': ['shux'],
    \ })
endif
```

LSP 提供完整功能：诊断、补全、跳转、悬停文档、重命名、格式化。

#### 2.12 缩进精化

**现状**：基于 cindent，部分 SHUX 结构缩进不准
**问题**：
- `match` arm 的 `=>` 后缩进
- `struct_literal` 多行字段缩进
- 链式方法调用 `.method()` 换行缩进
- `if` 表达式多行条件缩进

**实现**：自定义 `indentexpr`

```vim
" indent/shux.vim 用 indentexpr 替代 cindent
setlocal indentexpr=ShuxIndent()

function! ShuxIndent() abort
  let prev = getline(prevnonblank(v:lnum - 1))
  let curr = getline(v:lnum)

  " match arm => 后缩进
  if prev =~ '=>'
    return indent(v:lnum - 1) + &shiftwidth
  endif

  " struct literal { 后缩进
  if prev =~ '{\s*$'
    return indent(v:lnum - 1) + &shiftwidth
  endif

  " } 回退
  if curr =~ '^\s*}'
    return indent(v:lnum - 1) - &shiftwidth
  endif

  " 默认用 cindent
  return cindent(v:lnum)
endfunction
```

#### 2.13 状态栏集成

**现状**：无
**目标**：状态栏显示当前函数名 / struct 名

**实现**：`autoload/shux.vim` 提供函数，配合 lightline / airline

```vim
function! shux#CurrentFunction() abort
  let pos = getpos('.')
  let line = search('^\s*\(export\s\+\)\?function\s\+\zs[a-zA-Z_][a-zA-Z0-9_]*', 'bnW')
  if line == 0 | return '' | endif
  return 'fn:' . matchstr(getline(line), 'function\s\+\zs[a-zA-Z_][a-zA-Z0-9_]*')
endfunction
```

## 六、维护原则

遵循 AGENTS.md「禁止双权威」原则：

1. **关键字列表**：权威来源 `lexer.x` 的 `try_keyword`，Vim syntax 和 tree-sitter grammar 同源派生
2. **语法规则**：与 `parser.x` 的解析逻辑同步，不猜测语法
3. **补全候选**：原始类型/关键字从 `lexer.x` 提取，buffer-local 符号从当前文件扫描
4. **LSP 接入后**：诊断/补全/跳转的权威转为 `shux --lsp`，Vim 插件仅做 UI 层
